# Badminton Buddies Match Setter

A responsive doubles badminton match setter for six members.

## Features

- Creates fair doubles combinations for six players.
- Prevents players from playing more than two continuous games when possible.
- Tracks 21-point games.
- Shows the latest winner with a celebration panel.
- Stores only the current day's scores and clears old saved data daily.
- Exports today's matches and member map as CSV.

## Run Locally

Open `index.html` in a browser, or serve the folder:

```bash
python -m http.server 4173 --bind 127.0.0.1
```
